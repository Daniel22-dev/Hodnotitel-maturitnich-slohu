# Nahrání Hodnotitele 1.4.0

Rozbalte distribuční ZIP a nahrajte jeho obsah přímo do kořene repozitáře Hodnotitele. Nechte soubory přepsat, potvrďte commit a vyčkejte na zelené Actions/Pages.

Verze 1.4.0 eviduje anonymní technické počty zpracovaných slohů, včetně dávkových a částečných výsledků. Text studentské práce ani výsledek hodnocení se neukládá do telemetrie.
