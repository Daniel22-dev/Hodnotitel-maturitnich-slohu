# Hodnotitel maturitních slohů 1.4.0

- Telemetrie počítá jednu technickou položku `essay-evaluation` za každou práci předanou ke zpracování.
- Dávkové zpracování umí zapsat počet úspěšných, chybných a zrušených prací, včetně částečného výsledku.
- Batch API zapíše metriku až při skutečném dokončení, chybě nebo zrušení a brání dvojímu započtení.
- Text slohu, hodnocení, jméno studenta, třída ani jiné údaje se do souhrnu neukládají.
- Interaktivní manuál je z měření používání vyloučen.
